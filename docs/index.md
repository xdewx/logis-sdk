## 安装

```bash
pip install logis-sdk=={{ sdk_version() }} -i https://pypi.org/simple/
```

## 项目简介

本SDK目录结构如下：

```text

logis
├── alg        # 算法模块，包含寻路等算法
├── biz        # 业务模块，包含仿真大师等业务
├── conf       # 配置模块，包含配置文件等
├── ctx        # 上下文模块，包含上下文信息等
├── data_type  # 数据类型模块，包含数据类型定义等
├── decorator  # 装饰器模块，包含装饰器定义等
├── event      # 事件模块，包含事件定义、事件总线等
├── i18n       # 国际化模块，包含国际化定义等
├── iface      # 接口模块，包含接口定义等
├── logging    # 日志模块，包含日志定义等
├── math       # 数学模块，包含数学函数定义等
├── metric     # 指标模块，包含指标定义等
├── mq         # 消息队列模块，包含消息队列定义等
├── patch      # 补丁模块，包含补丁定义等
├── simpy      # 对simpy的扩展
├── task       # 任务模块，包含任务定义等
└── util       # 工具模块，包含工具函数定义等
```
